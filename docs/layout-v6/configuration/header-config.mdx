# Header

The Header component must be a direct child of the Root component. It is used to display the [header](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/header) of the layout.

## Adjusting height

Use `applyHeaderStyles` to set the height of the Header. The value can be a string with valid CSS units or a responsive object with [Material UI breakpoint](https://mui.com/material-ui/customization/breakpoints/#default-breakpoints) as keys.

You can set the CSS [position](https://developer.mozilla.org/en-US/docs/Web/CSS/position) as `relative` or `sticky` with the `sx` prop too.

export function DemoHeight() {
  return (
    <Root sx={{ minHeight: "20vh" }}>
      <Header
        sx={{
          ...applyHeaderStyles({
            height: { xs: 48, sm: 64, md: 72 },
          }),
          p: 2,
          position: "sticky",
          borderBottom: 1,
          borderColor: "divider",
        }}
      >
        Header
      </Header>
    </Root>
  );
}

<section className="demo-container">
  <DemoHeight />
</section>

## Stay above EdgeSidebar

By default, if an [EdgeSidebar](/docs/layout-v6-configuration-edge-sidebar--docs) is present, it will be placed beside the [Header](/docs/layout-v6-configuration-header--docs).
If you want the Header to be above the EdgeSidebar, set `fullWidth: true` in the `applyHeaderStyles` function:

export function DemoHeaderAboveEdgeSidebar() {
  return (
    <Root sx={{ ...applyRootStyles({ height: "30vh" }) }}>
      <Header
        sx={{
          ...applyHeaderStyles({ fullWidth: true }),
        }}
      >
        Header
      </Header>
      <EdgeSidebar sx={{ p: 2 }}>EdgeSidebar</EdgeSidebar>
      <Content>Content</Content>
      <Footer>Footer</Footer>
    </Root>
  );
}

<section className="demo-container">
  <DemoHeaderAboveEdgeSidebar />
</section>
