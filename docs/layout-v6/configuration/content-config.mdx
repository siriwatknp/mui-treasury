# Content

The Content component must be a direct child of the Root component.
It is used to display the `<main>` content of the layout.

The Content component has no configuration.

```jsx
import { Content, Root } from "path/to/layout-core-v6";

<Root>
  <Content>Content</Content>
</Root>;
```
