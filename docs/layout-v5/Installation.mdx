## Installation

MUI Treasury no longer provides npm packages. Instead, you can run the CLI to pull the source code to your project.

```sh
npx mui-treasury@latest clone layout-core-v5
```

By default, the source code will be cloned to `src/mui-treasury/layout-core-v5` unless you specify a `dir` in `<root-directory>/mui-treasury.config.js` file.

## Basic Usage

Import the layout components from the folder you cloned to your project.

```jsx
import {
  Content,
  EdgeSidebar,
  EdgeTriggger,
  Footer,
  Header,
  SidebarContent,
} from "path/to/layout-core-v5-v5";
```

The layout components support various type of application, for example:

- [Blog](/docs/layout-v5-tutorials-blog--docs)
- [Dashboard](/docs/layout-v5-tutorials-dashboard--docs)
- [Analytics](/story/layout-v5-app-analytics--analytics)
